This application was created in 2008 to show my bride's graphic arts portfolio. At that time, Flash was the predominant tool to make highly interactive websites. However, this app showed that DHTML libraries—like YUI used here—could achieve equal functionality using native browser technologies.
Art Gallery
Each portfolio tab represents a category and displays related pieces. (For this demonstration, arbitrary categories and a limited set of images are used.) Highlighting the tabs on page load shows the user what to do next: click a tab. Until the user takes action, select portfolio pieces are shown in a slideshow.
